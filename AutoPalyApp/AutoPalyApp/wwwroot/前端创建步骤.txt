npm install axios --save
项目引入，可以设置为全局引用：
import axios from 'axios'
// 将 axios 绑定到 Vue 原型上，方便全局使用
Vue.prototype.$axios = axios;
// 配置 axios 默认的根路径
axios.defaults.baseURL = 'http://localhost:5000';
